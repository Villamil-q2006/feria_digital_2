# Feria Digital – El Progreso

## Descripción del Proyecto
Esto es un mini pagina web desarrollado con HTML, CSS y JavaScript para mostrar los productos y logros de la vereda El Progreso.

## Cómo ejecutar
Abrir el archivo index.html en el navegador.

## Funcionalidad JavaScript implementada
- Botón "Ver más" que muestra u oculta información.
- Validación de formulario:
  - Nombre obligatorio.
  - Celular de 10 dígitos.
  - Mensaje mínimo 10 caracteres.
- Evidencia en consola con console.log().
